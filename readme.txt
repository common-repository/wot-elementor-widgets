Unlock powerful Elementor widgets for tabs, sliders, and more, designed for seamless customization and enhanced web design.

== Description ==

Wot Elementor Widgets is a lightweight collection of Elementor widgets with customizable features like given below features.

== Features ==

#### TabFlex
TabFlex is a multi-purpose tabbing plugin for Elementor that provides many customization features when developers need to implement a tabbing section on their website.
- Enhance your website with dynamic tabbed content.
- Horizontal tabs design.
- Vertical tabs design.
- Easy management of tabs.
- No interference with other Elementor widgets and plugins.

#### Tab Slider
The Tab Slider widget allows you to create beautiful tabbed content sliders effortlessly.
- Smooth transitions between tabs.
- Fully responsive design for optimal viewing on all devices.
- Customizable designs.
- Easy drag-and-drop interface for quick setup.
- Seamless integration with Elementorâ€™s existing styles.

#### Testimonials Slider
The Testimonials Slider widget enables you to showcase customer feedback and testimonials in a visually appealing manner.
- Display multiple testimonials with smooth sliding animations.
- Customizable layouts for unique designs.
- Option to add customer images for enhanced credibility.
- Control over slide duration and transition effects.
- User-friendly settings for easy management.

#### Post Slider
The Post Slider widget allows you to showcase your blog posts in a dynamic slider format.
- Automatically pulls in posts from your blog.
- Customizable layout and display options for post thumbnails and titles.
- Show posts categories or author details.
- Smooth sliding animations for engaging user experience.
- Simple settings for easy integration into your website.

== Installation ==

**1. Download the Archive:**
- Obtain the plugin archive from the source.

**2. Upload Method 1 (WordPress Dashboard):**
- Navigate to Plugins > Add New > Upload page.
- Upload the zip file.

**&hellip; OR &hellip;**

**3. Upload Method 2 (FTP):**
- Unpack the archive.
- Upload the unpacked folder using your favorite FTP client to the /plugins/ directory of your WordPress installation.

**4. Activation:**
- Once the installation is complete, activate the plugin.

== Usage ==

After installing Wot Elementor Widgets, you can access the additional widgets and tools directly within the Elementor interface. Simply drag and drop them onto your canvas to incorporate them into your designs.
